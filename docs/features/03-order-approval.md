# 기능 명세: 주문 승인/거절

> [← PRD.md로 돌아가기](../PRD.md)

접수된 `RESERVED` 상태의 주문 목록을 확인하고, 특정 주문에 대해 승인 또는 거절합니다.

## 주문 승인

재고 상황에 따라 자동 처리됩니다.

| 조건 | 처리 결과 | 전환 상태 |
|------|-----------|-----------|
| 재고 충분 | 재고에서 즉시 출고 대기 상태로 전환 | `RESERVED` → `CONFIRMED` |
| 재고 부족 | 생산 라인에 자동 등록 후 생산 대기 | `RESERVED` → `PRODUCING` |

## 주문 거절

즉시 거절 처리됩니다.

| 조건 | 처리 결과 | 전환 상태 |
|------|-----------|-----------|
| 거절 선택 | 모니터링에서 제외되는 비정상 종료 상태 | `RESERVED` → `REJECTED` |
