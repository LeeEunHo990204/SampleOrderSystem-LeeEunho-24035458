# 기능 명세: 모니터링

> [← PRD.md로 돌아가기](../PRD.md)

담당자가 현재 시스템의 상태를 한눈에 파악할 수 있도록 구성합니다.

## 주문량 확인

- 현재 상태별(RESERVED / CONFIRMED / PRODUCING / RELEASE) 주문 수 확인
- `REJECTED`는 유효한 주문이 아니므로 제외

## 재고량 확인

- 각 시료별 현재 재고 수량 확인
- 주문 대비 재고 수량에 따른 상태 표기:

| 상태 | 조건 |
|------|------|
| **여유** | 주문 대비 재고 충분 |
| **부족** | 주문 대비 재고 수량 부족 |
| **고갈** | 수량이 0인 상태 |
