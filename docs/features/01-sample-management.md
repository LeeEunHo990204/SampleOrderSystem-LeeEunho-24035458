# 기능 명세: 시료 관리

> [← PRD.md로 돌아가기](../PRD.md)

시료(Sample)는 이 시스템의 가장 기본이 되는 단위입니다. 각 시료는 고유한 이름과 속성을 가지며, 시스템에 등록된 시료만 주문 가능합니다.

## 기능 목록

| 기능 | 설명 |
|------|------|
| 시료 등록 | 새로운 시료를 시스템에 추가. 입력값: 시료 ID, 이름, 평균 생산시간, 수율 |
| 시료 조회 | 등록된 모든 시료 목록 및 현재 재고 수량 표시 |
| 시료 검색 | 이름 등 속성으로 특정 시료 검색 |

> **수율** = 정상적인 시료 / 총 생산 시료 (예: 100개 생산 중 정상 90개 = 0.9)
